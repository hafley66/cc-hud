## Key Technical Findings
- egui_table 0.8.0 has NO flex-grow weights. Column::auto_size distributes surplus EQUALLY in absolute pixels.
- Fixed columns MUST use range(w..=w) or they default to 4.0..=INF and steal surplus space from flex columns.
- AutoSizeMode::OnParentResize triggers redistribution when container width changes.
- auto_size_this_frame(true) is content-based sizing (measures cell content), separate from space-distribution.
- resizable(bool) only controls user drag-resize, NOT auto_size behavior.
