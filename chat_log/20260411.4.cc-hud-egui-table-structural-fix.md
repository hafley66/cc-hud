## Problem/Context
- 3_legend_table.rs had duplicate function definitions and borrow checker conflicts
- egui_table 0.7→0.8 version compatibility issues
- Session data has full event timeline in events: Vec<Event> (ApiCall, ToolUse, SkillUse, ReadFile, AgentSpawn, Compaction)
- Need to add 3-level nesting (Group → Session → Events) with expandable event grid

## Solution/Approach
Fixed structural issues by:
- Removing duplicate cell_swatch, cell_name_stats, cell_detail_button, cell_timeline, render_subagent_entry definitions
- Adding #[derive(Clone)] to TableRow enum
- Fixing borrow checker by cloning SessionData before mutable calls
- Updating deprecated APIs (Frame::none → Frame::NONE, rounding → corner_radius, Margin::same(6.0) → Margin::same(6))
