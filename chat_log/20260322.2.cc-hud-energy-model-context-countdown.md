- `ModelTier` (Haiku/Sonnet/Opus) with `from_model_str()`

- `TokenCounts` { input, output, cache_read, cache_create }

- `SessionEnergy` accumulator with `add_call()` per ApiCall event
- `EnergyConfig` with defaults: $0.17/kWh, us-east-1 (350 gCO2/kWh), PUE 1.2, 4090 GPU, US avg solar

- Haiku: 0.15 J, Sonnet: 0.9 J, Opus: 5.0 J
- Input (prefill): 5% of output. Cache hits: 0.
- Local 4090 reference: 21.43 J/tok, 5.95 kWh/M tokens

## Research Findings (from 4 Sonnet agents)

### Per-token energy
- Anthropic publishes nothing. Stanford FMTI confirmed zero environmental disclosure.
- Google: 0.24 Wh/median Gemini prompt (peer-reviewed Aug 2025). OpenAI: 0.34 Wh/query.
- Third-party Opus estimate: 4.05 Wh per ~400-token exchange (weak provenance).
- Decode dominates: >=96% of total energy. Prefill <=3.4%.
- ±3-5x uncertainty on absolutes, inter-tier ratios more stable.

### Carbon intensity
- EPA eGRID 2023 for US subregions. Electricity Maps for live data (free tier: 1 zone, 50 req/hr).
- CO2.js/Ember for static annual averages (no API needed). World avg 442 gCO2/kWh.
- AWS PUE ~1.2 (estimated), Google 1.09 (published).
- Solar: 400W panel, US avg 17% capacity factor = 68W continuous.

### Local pricing
- US avg $0.17/kWh (EIA 2025). Range $0.11 (ND) to $0.42 (HI).
- EIA Open Data API: free, monthly, by state. No hard rate limits.
- 4090 running 70B: ~$0.42/M tokens electricity. Opus API = ~36x markup.
- DeepSeek API ($0.28/M) cheaper than self-hosting electricity.

### Anthropic infrastructure
- AWS primary (Project Rainier: PA, IN, MS). GCP secondary (1M TPUs).
- Bedrock inference: us-east-1, us-east-2, us-west-2 + EU/AP regions.
- No sustainability report, no per-token energy, no carbon targets.
