## Problem/Context
The legend panel jumps straight into scrollable session group rows with no aggregate view. User wants a fixed stats strip above the ScrollArea showing cross-session cost patterns -- useful for showing coworkers what sessions actually cost.

## Solution/Approach
Approved plan at: `~/.claude/plans/quizzical-enchanting-sutton.md`

Insert fixed-height stats section (~28px, two rows of 9px mono) between `panel_frame().show()` and `ScrollArea::vertical()` in `draw_big()`.

**Row 1 -- per-session cost breakdown:**
- `avg $/sesh` -- mean total_cost_usd across visible sessions
- `$/200k projected` -- `(avg_cost / avg_tokens) * 200_000` -- projected cost if every session hit context limit
- `>200k: N/M` -- sessions exceeding 200k tokens with avg cost for those
- `$/1M tok` -- aggregate cost per million tokens

**Row 2 -- agent cost breakdown:**
- `agents: X%` -- total subagent cost / total session cost
- `avg $X/agent` -- mean subagent cost
- `X.X ag/sesh` -- mean subagent count per session

**Per legend row:** append agent cost to stats line if session has subagents.
