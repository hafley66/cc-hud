# Tooltip Column Headers Explainer

## Cost Mode

| Column | Tooltip Explanation |
|--------|---------------------|
| `in` | Fresh input tokens (non-cached) |
| `in(read)` | Cached input tokens (read from cache) |
| `in(write)` | Tokens written to prompt cache |
| `out` | Output tokens (non-thinking) |
| `out(think)` | Thinking/reasoning tokens |
| `turn` | Cost change this turn |
| `cumul` | Cumulative session total |

## Tokens Mode

| Column | Tooltip Explanation |
|--------|---------------------|
| `in` | Fresh input tokens |
| `in(read)` | Cached input tokens |
| `in(write)` | Tokens written to cache |
| `out` | Output tokens |
| `out(think)` | Thinking tokens |
| `cumul` | Cumulative tokens |

## Energy Mode

| Column | Tooltip Explanation |
|--------|---------------------|
| `Wh` | Facility energy (Wh) |

## Water Mode

| Column | Tooltip Explanation |
|--------|---------------------|
| `mL` | Total water (mL) |

## Budget Mode

| Column | Tooltip Explanation |
|--------|---------------------|
| `cost` | Turn cost |
| `+delta` | Cost change |
| `total` | Cumulative total |
